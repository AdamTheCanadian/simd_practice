## Notes

- __mm_set_ps or _mm_add_ps
    - called intrinsics 
    - All compile to a single instruction
    
- _mm_sub_ps
    - Subtract
- _mm_mul_ps
    - multiply
- _mm_div_ps
    - Division
- _mm_sqrt_ps
    - sqrt
- _mm_rcp_ps
    - reciprocal

- SSE
  - 128 bit level instructions
- AVX
  - Added 256 (and maybe more) bit level instructions
